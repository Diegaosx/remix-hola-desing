# Brand assets

Suba aqui:

- `logo.png` — carimbo oficial (https://holadesign.com.br/wp-content/uploads/2021/07/carimbo-768x790.png)
- `logo.svg` — versão vetorial (opcional, preferida sempre que possível)

Enquanto `logo.png` não estiver presente, a nav usa o monograma `¡H!` estilizado
(Newsreader italic) — mesmo comportamento dos HTMLs exportados do Stitch.

O favicon/apple-icon atuais (`src/app/icon.svg` e `src/app/apple-icon.svg`) também
são baseados no monograma `¡H!` sobre fundo `#1d1233`. Ao subir o PNG oficial,
renomeie (ou copie) para `src/app/icon.png` (Next 15 descobre automaticamente) e
os SVGs serão substituídos.
